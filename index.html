  • Introduction

Exact Capture is a high-rate, lossless packet capture solution for ExaNIC network adapters. The system is fully open source and designed for performance as well as ease of configuration. It can be used with any ExaNIC network adapter, and is optimised for use with ExaDISK high speed NVMe SSDs. The system can be deployed on any suitably powerful server system.
